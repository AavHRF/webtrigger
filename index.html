WebTrigger

WebTrigger is a suite of NationStates tools for triggering and conducting R/D.

 Downloads

Snapsheet

Snapsheet is a tried and true tool, getting its origins from the old update tool Spyglass. However, Snapsheet is fully cloud-based, enabling anybody to use it, regardless of platform.

Go to Tool

QuickDraw

QuickDraw is a tool for finding targets and trigger regions, and it goes off parameters you specify. Much easier than doing it manually! It works in conjunction with Snapsheet or Spyglass.

Go to Tool

Report Generator

A quick tool for listing who updated, and what targets you hit in a text file. Perfect for helping track R/D actions over time. Generates listings and places them in a "reports.txt" file in the working directory. Not cloud based.
